Overview of MongoDB

MongoDB consists of a set of databases. Each database again consists of Collections. Data in MongoDB is stored in collections. The below figure depicts the typical database structure in MongoDB.

More: https://www.studytonight.com/mongodb/overview-of-mongodb

Open Serve

mongod –dbpath=”C:\MongoDB\data\db”

Use

mongo

db
get the name of the current database as result

show dbs
see the list of all the databases in MongoDB

admin 0.000GB
config 0.000GB
elm 0.001GB
local 0.000GB
test 0.000GB —> Default

use databaseName
create a new database with the name databaseName

db.dropDatabase()
drop current database

CRUD

db.collection.insert( Document )
insert operation into a collection of a document(if the collection not exist,create it directly)

db.student.insert({
… regNo: “3014”,
… name: “Test Student”,
… course: {
… courseName: “MCA”,
… duration: “3 Years”
… }
… })

db.collection.insertOne({ … }) — inserts a single document into a collection.
db.collection.insertMany([ … ]) — insert multiple documents into a collection. Pass an array of documents to the method.

db.student.update({ “regNo”: “3014” },{ $set: { “name”:”Tadm” }})
WriteResult({ “nMatched” : 1, “nUpserted” : 0, “nModified” : 1 })

db.student.find().pretty()
{
“_id” : ObjectId(“5e259f40a10a41f84a41b974”),
“regNo” : “3014”,
“name” : “Tadm”,
“course” : {
“courseName” : “MCA”,
“duration” : “3 Years”
},
“address” : {
“city” : “Bangalore”,
“state” : “KA”,
“country” : “India”
}
}

The find() Method
To query data from MongoDB collection,display all the documents in a non-structured way.

The pretty() Method
To display the results in a formatted way

db.collection_name.remove({“fieldname”:”value”})

db.student.remove({“regNo” : “3014”})
